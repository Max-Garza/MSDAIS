$$Single \ \ server \ \ |\ \ \rho = \frac{\lambda}{\mu}$$
$$Multiple \ \ servers \ \ |\ \ \rho = \frac{\lambda}{c \mu}$$
rho = server utilization rate
lambda = mean arrival rate
mu = mean service rate
c = number of servers

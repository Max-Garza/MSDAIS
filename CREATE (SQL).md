A DDL command that allows one to create a relational database object
```SQL
-- Create a Database
>> CREATE DATABASE *database*

-- Create an Entity
>> CREATE TABLE *table* (
>> 	clmn1     datatype     constraint,
>> 	clmn2     datatype     constraint,
>> 	PRIMARY KEY (clmn1),
>> 	FOREIGN KEY (clmn 2) REFERENCES *table*
>> 	CONSTRAINT *constraint*
>> 	)
```
